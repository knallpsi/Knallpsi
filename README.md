## 👾 Обо мне
Я специализируюсь на низкоуровневом программировании, разработке игровых движков и встраиваемых системах (Embedded).

- ⚙️ **Фокус:** Low-level архитектура, графика, микроконтроллеры, оптимизация.
- 🛠 **Подход:** Пишу код, который работает быстро, надежно.
- 💡 **Интересы:** Рендеринг, системное программирование, hardware.
